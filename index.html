Object detection

A real-time object detection and localization system that utilizes color-based segmentation and depth estimation with Realsense depth camera D435. Implemented in ROS2 , the system detects and tracks objects of specific colors while estimating their 3D positions in real-world coordinates. This work contributes to the field of computer vision and robotic perception by offering a robust and efficient approach for real-time object recognition and localization in robotic systems.

Feedback control

This code runs a feedback control algorithm to stabilise the position of a UAV with external reference position and yaw inputs provided by a user. The implemented algorithm follows a PID Control structure. After parameter tuning, the UAV achieves smooth and precise position tracking. Through this process, I have learned that while implementing the algorithm itself is straightforward, fine-tuning the parameters is the most challenging aspect. Achieving optimal performance requires extensive testing, patience, and iterative adjustments.

Autonomous Fetching Robot

This project aims to integrate a robotic arm with an autonomous vehicle. The movement of the autonomous vehicle is controlled by varying the motor speeds using PWM signals from an Arduino. The robotic arm utilizes SG90 servo motors, with PWM signals controlling the rotation angles, and the motion is simulated using forward kinematics in MATLAB. For visual recognition, TensorFlow is used to train the system for object identification.
